Hangman Game with Tkinter
This is a simple Hangman game written in Python using the Tkinter library for GUI. The game has a word list of 10 words and allows the player to guess letters until they either correctly guess the word or reach 6 incorrect attempts.

Features
Display the word as dashes to be filled in as the player guesses the letters
Display the incorrect letters that the player has guessed
Display the remaining attempts the player has
Display a message if the player wins or loses the game
The game also provides clues after two wrong attempts
A button for submitting the player's guess
Requirements
Python 3.x
Tkinter library
How to Run the Game
Clone or download the repository to your local machine
Install Tkinter library if not already installed
Open a terminal or command prompt in the directory where the game is located
Run the following command to start the game:
Copy code
python hangman.py
Follow the instructions in the GUI to play the game
Customizing the Game
You can add or change the words in the word list by modifying the word_list variable in the code.
You can adjust the number of incorrect attempts before the game ends by changing the value of the MAX_ATTEMPTS constant.
